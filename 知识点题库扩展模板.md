## 📋 一、知识点添加模板

### 基础模板（复制使用）

```javascript
// 在 KNOWLEDGE_BASE 对应分类的 topics 下添加：

newTopicKey: {
  id: 'newTopicKey',                    // 唯一标识符（驼峰命名）
  name: '知识点名称',                    // 中文名称
  difficulty: 3,                        // 难度 1-5
  formulas: [                           // 相关公式列表
    '公式1',
    '公式2',
    '公式3'
  ],
  description: '知识点的简要说明，不超过50字',
  relatedTypes: ['uniform'],            // 关联的题型：uniform/projectile/circular/collision/magnetic/astrodynamics
  keywords: [                           // 关键词（用于AI识别）
    '关键词1', '关键词2', '关键词3'
  ],
  examples: [                           // 典型例题（3-5道）
    {
      title: '例题标题',
      description: '完整的题目描述',
      params: {                         // 参数（根据题型不同）
        v0: 0,
        a: 2,
        time: 3
      },
      type: 'uniform'                   // 题型类型
    }
  ]
}
```

## 📝 三、例题添加模板

### 3.1 匀变速直线运动例题模板

```javascript
{
  title: '例题标题（简短概括）',
  description: '一辆汽车从静止开始以2m/s²的加速度做匀加速直线运动，求：(1) 5秒后的速度 (2) 5秒内的位移',
  params: {
    v0: 0,      // 初速度 (m/s)
    a: 2,       // 加速度 (m/s²)
    time: 5     // 时间 (s)
  },
  type: 'uniform',
  solution: {   // 可选：解题步骤
    steps: [
      '已知：v₀=0, a=2m/s², t=5s',
      '求：v=?, s=?',
      '由v=v₀+at得：v=0+2×5=10m/s',
      '由s=v₀t+½at²得：s=0+½×2×5²=25m'
    ],
    answer: 'v=10m/s, s=25m'
  }
}
```

### 3.2 抛体运动例题模板

```javascript
{
  title: '斜抛运动问题',
  description: '篮球以10m/s的速度、与水平方向成45°角投出，离地高度2m，求飞行时间和水平射程。(g=10m/s²)',
  params: {
    speed: 10,   // 初速度 (m/s)
    angle: 45,   // 角度 (度)
    height: 2    // 初始高度 (m)
  },
  type: 'projectile',
  solution: {
    steps: [
      '已知：v₀=10m/s, θ=45°, h=2m, g=10m/s²',
      '水平分速度：vₓ=v₀cosθ=10cos45°≈7.07m/s',
      '竖直分速度：vᵧ=v₀sinθ=10sin45°≈7.07m/s',
      '竖直方向：h+vᵧt-½gt²=0',
      '解得：t≈1.6s',
      '水平射程：x=vₓt≈11.3m'
    ],
    answer: 't≈1.6s, x≈11.3m'
  }
}
```

### 3.3 圆周运动例题模板

```javascript
{
  title: '匀速圆周运动',
  description: '摩天轮半径为20m，以角速度0.1rad/s匀速转动，求边缘一点的线速度和向心加速度。',
  params: {
    radius: 20,    // 半径 (m)
    angular: 0.1   // 角速度 (rad/s)
  },
  type: 'circular',
  solution: {
    steps: [
      '已知：r=20m, ω=0.1rad/s',
      '线速度：v=ωr=0.1×20=2m/s',
      '向心加速度：a=ω²r=0.1²×20=0.2m/s²',
      '或：a=v²/r=2²/20=0.2m/s²'
    ],
    answer: 'v=2m/s, a=0.2m/s²'
  }
}
```

### 3.4 碰撞问题例题模板

```javascript
{
  title: '弹性碰撞问题',
  description: '质量为2kg的小球以5m/s的速度与质量为1kg的静止小球发生弹性碰撞，求碰后两球的速度。',
  params: {
    m1: 2,    // 质量1 (kg)
    v1: 5,    // 速度1 (m/s)
    m2: 1,    // 质量2 (kg)
    v2: 0     // 速度2 (m/s)
  },
  type: 'collision',
  solution: {
    steps: [
      '动量守恒：m₁v₁+m₂v₂=m₁v₁\'+m₂v₂\'',
      '能量守恒（弹性碰撞）：½m₁v₁²+½m₂v₂²=½m₁v₁\'²+½m₂v₂\'²',
      '代入数据：2×5+1×0=2v₁\'+1v₂\'',
      '½×2×25+0=½×2v₁\'²+½×1v₂\'²',
      '解方程组得：v₁\'=3m/s, v₂\'=8m/s'
    ],
    answer: 'v₁\'=3m/s, v₂\'=8m/s'
  }
}
```

### 3.5 磁场运动例题模板

```javascript
{
  title: '带电粒子在磁场中运动',
  description: '电子(q=1.6×10⁻¹⁹C, m=9.1×10⁻³¹kg)以1×10⁶m/s的速度垂直进入0.5T的匀强磁场，求运动半径和周期。',
  params: {
    q: 1.6e-19,   // 电荷量 (C)
    v: 1e6,       // 速度 (m/s)
    B: 0.5        // 磁场强度 (T)
  },
  type: 'magnetic',
  solution: {
    steps: [
      '洛伦兹力提供向心力：qvB=mv²/r',
      '半径：r=mv/(qB)=9.1×10⁻³¹×10⁶/(1.6×10⁻¹⁹×0.5)',
      'r≈1.14×10⁻⁵m=11.4μm',
      '周期：T=2πr/v=2π×1.14×10⁻⁵/10⁶≈7.15×10⁻¹¹s'
    ],
    answer: 'r≈11.4μm, T≈71.5ps'
  }
}
```

### 3.6 天体运动例题模板

```javascript
{
  title: '地球卫星轨道计算',
  description: '计算距地面高度400km的国际空间站的运行周期和速度。(地球质量M=5.97×10²⁴kg, 地球半径R=6.37×10⁶m)',
  params: {
    M: 5.97e24,          // 中心天体质量 (kg)
    r: 6.77e6            // 轨道半径 (m) = R + h
  },
  type: 'astrodynamics',
  solution: {
    steps: [
      '已知：M=5.97×10²⁴kg, r=R+h=6.37×10⁶+4×10⁵=6.77×10⁶m',
      'G=6.67×10⁻¹¹N·m²/kg²',
      '万有引力提供向心力：GMm/r²=mv²/r',
      '速度：v=√(GM/r)≈7.67km/s',
      '周期：T=2πr/v≈5550s≈92.5分钟'
    ],
    answer: 'v≈7.67km/s, T≈92.5分钟'
  }
}
```

## 🎯 四、例题质量标准

### 4.1 优秀例题的特征

✅ **清晰明确**
- 题目描述完整，包含所有必要条件
- 物理量单位明确
- 问题具体，不模糊

✅ **难度适中**
- 符合知识点的难度等级
- 循序渐进，从简单到复杂

✅ **贴近实际**
- 使用真实场景（汽车、篮球、卫星等）
- 数值合理，符合实际情况

✅ **可视化友好**
- 适合动画演示
- 运动过程清晰可见

### 4.2 参数设置建议

**匀变速直线运动**：
- 初速度：0-30 m/s
- 加速度：1-10 m/s²
- 时间：1-10 s

**抛体运动**：
- 初速度：5-20 m/s
- 角度：15-75 度
- 高度：0-20 m

**圆周运动**：
- 半径：1-50 m
- 角速度：0.1-2 rad/s

**碰撞**：
- 质量：0.5-5 kg
- 速度：1-10 m/s

**磁场运动**：
- 电荷：1×10⁻¹⁹ C（电子）或更大
- 速度：10⁵-10⁷ m/s
- 磁场：0.1-2 T

**天体运动**：
- 中心天体质量：5.97×10²⁴ kg（地球）
- 轨道半径：6.7×10⁶-4×10⁷ m

```json
    }
  ]

```

### 6.3 验证工具

创建验证函数检查例题格式：

```javascript
function validateExample(example) {
  const required = ['title', 'description', 'params', 'type'];
  return required.every(field => example.hasOwnProperty(field));
}
```
