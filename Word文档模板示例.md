# Word 文档模板示例

将以下内容复制到 Microsoft Word 中，保存为 `.docx` 格式即可导入：

---

一、力学

1. 匀变速直线运动
公式：v = v₀ + at
公式：s = v₀t + ½at²
公式：v² - v₀² = 2as
描述：物体在恒定加速度作用下的直线运动，包括匀加速和匀减速两种情况

例题1：汽车刹车问题
一辆汽车以20m/s的速度行驶，突然刹车，加速度大小为5m/s²，求刹车后4秒的速度和刹车距离。

例题2：火车加速启动
火车从静止开始匀加速启动，加速度为2m/s²，求5秒后的速度和位移。

2. 抛体运动
公式：x = v₀cosθ·t
公式：y = h + v₀sinθ·t - ½gt²
公式：v_x = v₀cosθ
公式：v_y = v₀sinθ - gt
描述：物体在重力作用下的曲线运动，包括平抛、斜抛等

例题：篮球投篮
篮球以10m/s的速度、45°角投出，离地高度2m，求飞行时间和射程。

3. 圆周运动
公式：v = ωr
公式：a = v²/r = ω²r
公式：T = 2π/ω
描述：物体做圆周运动的运动学和动力学分析

例题：摩天轮运动
摩天轮半径20m，角速度0.1rad/s，求边缘的线速度和向心加速度。

二、电磁学

1. 磁场中的带电粒子运动
公式：F = qvB
公式：r = mv/(qB)
公式：T = 2πm/(qB)
描述：带电粒子在磁场中受洛伦兹力作用的圆周运动

例题：电子在磁场中运动
电子以1×10⁶m/s速度垂直进入0.5T的匀强磁场，求运动半径和周期。

2. 电磁感应
公式：E = BLv
公式：Φ = BS
公式：E = -dΦ/dt
描述：导体在磁场中运动产生感应电动势的现象

例题：导体棒切割磁感线
长度为0.5m的导体棒以2m/s的速度垂直切割磁感应强度为0.8T的匀强磁场，求感应电动势。

三、天体物理

1. 万有引力与卫星运动
公式：F = GMm/r²
公式：v = √(GM/r)
公式：T = 2π√(r³/GM)
描述：行星、卫星等天体在万有引力作用下的运动规律

例题：地球卫星轨道
计算距地面400km高度的卫星运行周期和速度。（已知地球质量M=5.97×10²⁴kg，地球半径R=6.37×10⁶m）

## 使用说明

### 如何创建你自己的知识点文档

1. **打开 Microsoft Word**
2. **复制上面的模板**（从"一、力学"开始到最后）
3. **修改为你自己的知识点内容**
   - 修改分类名称（一、二、三...）
   - 修改知识点名称（1. 2. 3.）
   - 修改公式、描述、例题
4. **保存为 `.docx` 格式**
   - 文件 → 另存为 → 文件类型选择"Word文档 (.docx)"
5. **导入到系统**
   - 打开 index1.0.3.html
   - 滚动到"物理知识点库"
   - 点击"📥 导入知识点"按钮
   - 选择你的 .docx 文件

### 格式要点

✅ **必须保留的格式：**
- 一、二、三...（大类标题）
- 1. 2. 3.（知识点标题）
- "公式："标记（每行一个公式）
- "描述："标记（可选）
- "例题"或"例题1"等（例题标题）

❌ **不要使用的格式：**
- Word的"标题1、标题2"样式（会被转换为纯文本）
- 复杂的表格（可能无法识别）
- 图片（会被忽略）
- 过于复杂的嵌套结构
