### 3. 圆周运动 (circularMotion)

```javascript
// 添加到 circularMotion 的 examples 数组中
```

## 🎯 使用说明

### 第一步：打开代码文件

打开 `index1.0.3.html` 文件，找到 `KNOWLEDGE_BASE` 对象（约第1030行）

### 第二步：定位知识点

找到要添加例题的知识点，例如 `uniformMotion`：

```javascript
uniformMotion: {
  id: 'uniformMotion',
  name: '匀变速直线运动',
  // ... 其他属性
  examples: [
    // 👈 在这里添加新例题
    {
      title: '汽车刹车问题',
      // ... 现有例题
    },
    // 👇 复制上面的例题代码粘贴到这里
  ]
}
```

### 第三步：添加例题

复制上面对应知识点的例题代码，粘贴到 `examples` 数组的末尾：

```javascript
examples: [
  // 现有例题...
  {
    title: '汽车刹车问题',
    description: '...',
    params: { v0: 20, a: -5, time: 4 },
    type: 'uniform'
  },
  // 👇 粘贴新例题
  {
    title: '汽车加速超车',
    description: '一辆汽车以10m/s的速度行驶...',
    params: { v0: 10, a: 2, time: 6 },
    type: 'uniform'
  }
]
```

### 第四步：保存并测试

1. 保存文件
2. 刷新浏览器
3. 滚动到"物理知识点库"
4. 点击对应知识点
5. 查看新添加的例题
6. 点击例题测试可视化效果

## 🔧 批量添加技巧

### 技巧1：使用查找替换

如果要批量修改参数，可以使用编辑器的查找替换功能：

- 查找：`v0: 0`
- 替换为：`v0: 5`

### 技巧2：复制整个数组

如果要添加多个例题，直接复制整个 examples 数组：

```javascript
examples: [
  // 复制这里所有例题...
]
```

### 技巧3：自动格式化

添加完例题后，使用编辑器的格式化功能（通常是 `Shift+Alt+F`）自动整理代码格式。
